# Arduino-Nano_based-simple-Calculator
Here i designed the simple calculator using ardiuno nano ,keypad 4x4 ,lcd I2C, it complute simple arithmetic operators to compute the calculation.
Here the pin connection to interface the components --->

keypad(4x4) ---> NANO <------Lcd I2C
r1-------->  D2    A4 <------SDA
r2-------->  D3    A5 <------SCL
r3-------->  D4    5V <------VCC
r4-------->  D5    GND<------GND
c1-------->  D6
c2-------->  D7
c3-------->  D8
c4-------->  D9
